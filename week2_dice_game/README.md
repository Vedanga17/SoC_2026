# Week 2: React Dice Game 🎲

This is my Week 2 assignment for the course. It is an interactive, React-based dice game that tests your luck, tracks your stats, and saves your scores!

## Features
* **Interactive Gameplay:** Select a number and roll the dice. You win points for matching, and lose points for missing.
* **Live Statistics Dashboard:** Tracks total rolls, correct/wrong guesses, and calculates accuracy in real-time.
* **Session Leaderboard:** Saves your scores and ranks them when you end the game.
* **Responsive Layout:** Flexbox design that works perfectly on both desktop and mobile screens.

## Tech Stack
* React.js
* Vite
* Standard CSS3
* JavaScript (Hooks & State Management)
